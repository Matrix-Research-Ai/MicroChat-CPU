Fine-tune an existing checkpoint on your data in chat format. Best for instruction/conversational datasets.

1

Install MicroChat-CPU

git clone https://github.com/Matrix-Research-Ai/MicroChat-CPU.git
cd MicroChat-CPU
source .venv/bin/activate || uv sync --extra cpu && source .venv/bin/activate
2

Convert to chat format

# Loading dataset...
3

Edit chat_sft.py

Open scripts/chat_sft.py and add CustomJSON to the train_tasks list:

# Add this line to train_tasks (around line 165)
CustomJSON(filepath=r"C:\path\to\dataset"),
4

Run SFT

python -m scripts.chat_sft --device-type=cpu --num-iterations=100

Requires a pretrained checkpoint first. Run python -m scripts.base_train --depth=4 --num-iterations=50 if you don't have one.

5

Chat with the fine-tuned model

python -m scripts.chat_cli --model-tag=chatsft/d4
💡 Your dataset will be converted to chat format with user/assistant messages.
▸ Quick Smoke Test
cd MicroChat-CPU
source .venv/bin/activate
python -m scripts.cpu_train --synthetic --dry-run --depth=2

Downloads nothing, runs in 2 seconds. If this passes, everything is installed correctly.
