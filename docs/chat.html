⚙️ How to Run
1

Wait for SFT training

Run this in MicroChat-CPU directory. When it finishes, the model is ready.

.venv\Scripts\python -m scripts.chat_sft --device-type=cpu --model-tag=d6 --model-step=200 --num-iterations=50 --eval-every=-1 --chatcore-every=-1
2

Start the web server

.venv\Scripts\python -m scripts.chat_web --device-type=cpu --source=sft --model-tag=d6 --step=50 --port=8000
3

Or chat in terminal

.venv\Scripts\python -m scripts.chat_cli --source=sft --model-tag=d6 --step=50 --device-type=cpu
